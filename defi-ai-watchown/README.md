# DeFi AI Watchown 🤖🔍
An autonomous AI agent that monitors a DeFi wallet on the Sepolia testnet, detects anomalies, and generates risk reports using Web3.py and Claude AI.

## Setup
1. Copy `.env.example` to `.env` and add your API keys.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python agent/alert.py`
